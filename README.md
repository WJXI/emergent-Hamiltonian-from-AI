# Autonomous Emergence of Hamiltonian in Deep Generative Models

Official code, datasets, and pre-trained models for the paper **"Autonomous Emergence of Hamiltonian in Deep Generative Models"**. 


---

## 📂 Repository Structure

- **`SO3.ipynb`**: Contains the complete pipeline for model training and the algebraic analysis.
- **`Fig.ipynb`**: Generates all the plots presented in the paper.
- **`mini_af3_model.py`**: PyTorch implementation of the $O(3)$-equivariant "Mini-AF3" generative architecture.
- **`mini_af3_epoch_20.pt`**: The pre-trained model checkpoint. **All results, figures, and physical analyses in the paper are strictly based on this specific checkpoint.**
